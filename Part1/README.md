# PKGF Intelligence Emergence
## 12次元並行鍵幾何流における知性の創発シミュレーション

本プロジェクトは、**Parallel Key Geometric Flow (PKGF)** 理論を拡張し、知性を「結合力学系における安定アトラクタとしての創発現象」として実証するための実験環境です。

### 1. 研究の背景
本研究は、自然言語の幾何学的遷移を記述する PKGF（https://doi.org/10.5281/zenodo.19217632 ）を基盤としています。知性を単なる最適化計算ではなく、12次元多様体上の物理的な流動、内的緊張、および社会的相互作用の結節点として捉え直すことが核心的な目的です。

### 2. 知性の16要素
本シミュレーターは、以下の16要素を力学的に統合しています。
1. 意味場 / 2. 文脈場 / 3. 幾何学的計量 $g$ / 4. 変換作用素 $K$ / 5. 欲求 $D$ / 6. 倫理 $E$ / 7. 感情 $A$ / 8. 価値形成 / 9. 学習 / 10. 記憶 $H$ / 11. メタ認知的評価 / 12. メタ更新 / 13. 自己参照 / 14. 意識アトラクタ / 15. 戦略形成 / 16. 社会的結合

### 3. 実験フェーズ
シミュレーションを実行すると、自動的に以下の3つのフェーズが進行します。

- **Phase A: Basic Stability (基礎安定性)**
  - 単一エージェントによる PKGF 流動の収束と、行列式 $det(K)$ の保存を確認します。
- **Phase B: Social Deadlock (社会的膠着)**
  - 意思決定を行わない複数エージェントが、相互斥力によって原点付近で静止する「デッドロック」状態を再現します。
- **Phase C: Strategic Emergence (戦略的創発)**
  - 飢餓感、内的緊張、および他者予測に基づく「戦略的決断」を導入し、対称性の自発的破れ（役割分担）が創発する瞬間を観測します。

### 4. 実行方法

#### Python
```bash
python3 emergence_sim.py
```

#### Fortran
```bash
gfortran -O3 -o emergence_sim_fortran emergence_sim.f90
./emergence_sim_fortran
```

### 5. 出力とログ
実行結果は言語ごとに `logs/` ディレクトリに保存されます。
- `logs/python/` (Pythonの実行結果)
- `logs/fortran/` (Fortranの実行結果)

また、実験の最終的な科学的考察は `FINAL_REPORT.md` に記録されています。

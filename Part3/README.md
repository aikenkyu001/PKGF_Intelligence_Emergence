# PKGF Intelligence Emergence - Part 3
## 多様体の次元数が知能の闘争と安定に与える影響の検証

Part3では、エージェント数 $N$ と多様体の次元数 $DIM$ を同期（$N = DIM$）させ、幾何学的な自由度が社会的階層の創発プロセスにどのような変容をもたらすかを検証します。

### 1. 研究の焦点
- **次元の自由度**: 高次元空間（直交する自由度）が、エージェント間の物理的衝突の回避にどう寄与するか。
- **闘争の寿命**: 空間の広さが、Aggressive モード（闘争状態）の持続時間に与える影響。
- **次元的解消の定理**: 「制約（低次元）が闘争を生み、自由（高次元）が沈黙をもたらす」という幾何学的社会学の法則の立証。

### 2. 実行方法

#### Python
```bash
python3 Part3/emergence_sim.py
```

#### Fortran
```bash
gfortran -O3 -o Part3/emergence_sim_dynamic Part3/emergence_sim.f90
./Part3/emergence_sim_dynamic
```

### 3. 実験ケース
- **Case 1**: $N=4, DIM=4$ (低次元・高干渉)
- **Case 2**: $N=8, DIM=8$ (中次元)
- **Case 3**: $N=16, DIM=16$ (高次元・高自由度)

### 4. 主要な発見（次元のパラドックス）
実験の結果、高次元多様体（DIM=16）では、エージェントが直交次元を利用して互いを「幾何学的に回避」できるため、内的緊張が速やかに緩和され、社会は平和な二階層（沈黙）へと早期に収束することが確認されました。対照的に、低次元環境では衝突が不可避となり、闘争が永続化する傾向が示されました。
